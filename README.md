## Getting started

### Prerequisites

1. [Install Node.js](https://nodejs.org/en/download/)

### Installation

1. Clone the repository:

```shell script
git clone https://github.com/lena-my/hackaton-back.git
```

2. Install dependencies:

```shell script
npm install
```

3. Build the project:

```shell script
npm run start
```
